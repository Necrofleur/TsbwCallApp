Zum erstellen einer APK:

1. In Android Studio unter "Build"
   den Menüpunkt "Generate Signed Bundle / APK..." wählen,
2. die Option "APK" wählen (d.h. nicht Bundle),
3. den Keystore "stuff/release-key.keystore" verwenden,
   den Alias "release_key" wählen und bei beiden Passwörtern
   L8w5wqFAW5A4qzwLfxpejEyG einfügen,
4. Buildvariante "release" wählen und beide Signaturversionen aktivieren

Die erstellte APK wird dann im Verzeichnis "app/release/app-release.apk"
abgelegt.
